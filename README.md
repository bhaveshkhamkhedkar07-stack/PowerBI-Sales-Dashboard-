# 📊 Motorola Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze Motorola sales performance. It helps users monitor key business metrics, identify sales trends, and make data-driven decisions using dynamic visualizations.

## 🚀 Features

- 📈 Total Sales KPI
- 📦 Total Quantity Sold
- 💳 Total Transactions
- 💰 Average Sales Value
- 🗺️ Sales by City (Map Visualization)
- 📅 Monthly Sales Analysis
- 📱 Sales by Mobile Model
- ⭐ Customer Ratings Analysis
- 💳 Payment Method Distribution
- 📆 Day-wise Sales Analysis
- 🔍 Interactive Filters (Slicers)
  - Mobile Model
  - Payment Method
  - Brand
  - Day Name
  - Month

 
## 🛠️ Tools Used

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)


## 📊 Dashboard Insights

- Monitor overall sales performance.
- Compare sales across different mobile models.
- Analyze customer payment preferences.
- Track sales by city and month.
- Identify trends using interactive filters.
